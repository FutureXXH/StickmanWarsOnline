# StickmanWarsOnline
# 一个联机火柴人大战游戏 Demo

当前完成度 60%

拥有开房间以及匹配功能，可以配置角色携带的技能


客户端使用Unity C#制作

服务端使用自己写的C++与Lua架构（https://github.com/FutureXXH/TCPOnlineGameServer）  ,这里只放了Lua模块代码
